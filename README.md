# skills-library

A collection of Claude Code skills for AI agents, built around professional methodology and structured reasoning.

## Skills

| Skill | Description |
|-------|-------------|
| [workplan](./workplan/) | Hypothesis-driven research workplans using MBB strategy consulting methodology |

## Installation

Each skill is self-contained. Copy the skill directory into your Claude Code skills folder:

```bash
cp -r workplan/ ~/.claude/skills/workplan/
```
